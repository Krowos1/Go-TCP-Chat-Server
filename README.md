# Go TCP Chat Server

Educational TCP chat server written in Go.

## Features

- TCP server
- Multiple clients
- Goroutine per client
- Nicknames
- Broadcast messages
- `/users` command
- `/help` command
- `/quit` command
- `/whisper <name> <message>` private messages

## Run

```bash
go run .

## Connect

```bash
ncat localhost 8080
